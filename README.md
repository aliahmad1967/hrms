
# نظام إدارة الموارد البشرية (HR Management System)

نظام شامل لإدارة الموارد البشرية مبني باستخدام Streamlit وSQLAlchemy باللغة العربية مع دعم التخطيط من اليمين إلى اليسار (RTL).

[![Python](https://img.shields.io/badge/python-3.8%2B-blue)](https://www.python.org/)
[![Streamlit](https://img.shields.io/badge/streamlit-latest-red)](https://streamlit.io/)
[![SQLAlchemy](https://img.shields.io/badge/sqlalchemy-latest-orange)](https://www.sqlalchemy.org/)

## 🌟 الميزات

- **لوحة تحكم** مع إحصائيات وإحصائيات بيانية
- **إدارة الموظفين** (إضافة، تعديل، حذف)
- **تتبع الحضور والانصراف**
- **إدارة طلبات الإجازة** مع نظام موافقة
- **حساب الرواتب** وعرضها تلقائيًا
- **تقارير مفصلة** مع إمكانية التصدير
- **نظام مصادقة** مع صلاحيات مختلفة (مدير/موظف)
- **واجهة باللغة العربية** مع دعم RTL
- **تصميم محسن** مع دعم السمات (Light/Dark)
- **تصدير البيانات** إلى Excel (قريبًا)

## 🛠️ المتطلبات

- Python 3.8 أو أعلى
- pip (مدير الحزم لبايثون)

## 🚀 التثبيت

### 1. استنساخ المستودع (Clone the Repository):

افتح موجه الأوامر أو الطرفية (Terminal) وشغل الأمر التالي:

```bash
git clone https://github.com/YOUR_USERNAME/hrms.git
```

استبدل `YOUR_USERNAME` باسم المستخدم الخاص بك على GitHub.

### 2. التنقل إلى المجلد:

```bash
cd hrms
```

### 3. إنشاء بيئة افتراضية (Recommended):

من الأفضل استخدام بيئة افتراضية لعزل حزم المشروع.

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. تثبيت الحزم المطلوبة:

```bash
pip install -r requirements.txt
```

### 5. تشغيل التطبيق:

```bash
streamlit run app.py
```

سيتم فتح التطبيق تلقائيًا في متصفحك الافتراضي على `http://localhost:8501`.

## 👥 بيانات الدخول الافتراضية

- **مدير النظام (Admin):**
  - **اسم المستخدم:** `admin`
  - **كلمة المرور:** `admin123`
  - **الصلاحية:** الوصول الكامل إلى جميع الميزات.
- **موظف (Employee):**
  - **اسم المستخدم:** `employee`
  - **كلمة المرور:** `emp123`
  - **الصلاحية:** عرض الحضور والإجازات الخاصة فقط (الوظيفة الأساسية قيد التطوير).

## 📖 دليل المستخدم

### 1. تسجيل الدخول

- عند تشغيل التطبيق لأول مرة، ستظهر لك شاشة تسجيل الدخول.
- أدخل بيانات الاعتماد (اسم المستخدم وكلمة المرور) المناسبة.
- انقر على زر "🚪 دخول".
- إذا كانت البيانات صحيحة، سيتم توجيهك إلى الشاشة الرئيسية.

### 2. لوحة التحكم (Dashboard)

- بعد تسجيل الدخول بنجاح، ستظهر لك **لوحة التحكم**.
- تعرض هذه اللوحة ملخصًا سريعًا لحالة النظام:
  - عدد الموظفين الكلي.
  - عدد الموظفين الحاضرين اليوم.
  - عدد طلبات الإجازة المعلقة.
- توجد أيضًا مخططات بصرية (مثل Pie Chart) لتحليل توزيع الموظفين حسب الأقسام.

### 3. إدارة الموظفين (Employee Management)

- اختر "👥 إدارة الموظفين" من القائمة الجانبية.
- تحتوي هذه الشاشة على ثلاثة جداول (Tabs):
  - **عرض الموظفين:** يعرض قائمة بجميع الموظفين المسجلين مع تفاصيلهم.
  - **إضافة موظف:** يتيح لك إدخال معلومات موظف جديد (الاسم الكامل، رقم الهوية، الوظيفة، تاريخ التعيين، الراتب).
    - املأ النموذج واضغط "➕ إضافة موظف".
  - **تعديل/حذف:** يتيح لك اختيار موظف من القائمة وتعديل بياناته أو حذفه.
    - اختر الموظف من القائمة المنسدلة.
    - عدل البيانات في الحقول النصية.
    - اضغط "✏️ تحديث الموظف" أو "🗑️ حذف الموظف".

### 4. تتبع الحضور (Attendance Tracking)

- اختر "📅 تتبع الحضور" من القائمة الجانبية.
- تحتوي هذه الشاشة على جدولين (Tabs):
  - **تسجيل الحضور:** لتسجيل حضور أو غياب موظف يدويًا.
    - اختر الموظف من القائمة.
    - حدد التاريخ.
    - اختر الحالة (حاضر/غائب).
    - اضغط "✅ تسجيل الحضور".
  - **تقارير الحضور:** يعرض سجل الحضور المسجل سابقاً.

### 5. طلبات الإجازة (Leave Requests)

- اختر "📝 طلبات الإجازة" من القائمة الجانبية.
- تحتوي هذه الشاشة على جدولين (Tabs):
  - **تقديم طلب:** لتقديم طلب إجازة جديد (متوفر للمستخدمين ذوي صلاحيات الموظفين فقط في الإصدار الحالي).
    - اختر نوع الإجازة.
    - حدد تاريخ البدء والانتهاء.
    - اكتب السبب (اختياري).
    - اضغط "📤 إرسال الطلب". سيتغير status إلى "بانتظار الموافقة".
  - **إدارة الطلبات:** لعرض جميع طلبات الإجازة ومراجعة الطلبات المعلقة و الموافقة عليها أو رفضها (متوفر للمستخدمين ذوي صلاحيات المدير فقط).
    - ستظهر قائمة بجميع الطلبات مع الحالة (بانتظار الموافقة، موافق، مرفوض).
    - في قسم "الموافقة على الطلبات"، اختر الطلب المعلق من القائمة.
    - اضغط "✅ الموافقة" أو "❌ رفض" لتحديد القرار.

### 6. الرواتب (Salaries)

- اختر "💰 إدارة الرواتب" من القائمة الجانبية.
- تحتوي هذه الشاشة على جدولين (Tabs):
  - **عرض الرواتب:** يعرض قائمة برواتب الموظفين المسجلة.
  - **حساب الرواتب:** لحساب وحفظ راتب شهري لموظف.
    - اختر الموظف والشهر.
    - أدخل الراتب الأساسي والخصومات.
    - سيظهر "صافي الراتب" تلقائيًا.
    - اضغط "💾 حفظ الراتب".

### 7. تسجيل الخروج

- في الشريط الجانبي، انقر على زر "🚪 تسجيل الخروج".
